Vocabulary

Protocol
standardized set of rules and procedures that allows different devices and systems to communicate and exchange data with each other
Packet
Small, managable unit of data that travels over the internet and other networks
Header
IP/Network packet headeres with routing information like source and destination addresses
Payload
data being transmitted in a network packet that comes after the header
Router
a networking device that directs data packets between different computer networks, including the internet
Routing table
A data file in a router or networked computer that stores rules for directing data packets across a network
