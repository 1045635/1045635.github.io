How do I get into vim?

  1. make sure you have a file to edit
  2. type vim (file name)
  3. Yay! You're in!

How do I get out of vim?

  1. press the esc key
  2. type :wq then press enter

Basics of editing/using VIM:

Editing:

When you first enter vim you can't type or edit anything.

To edit vim, press i

Manuvering Around the Page

The main keys used to move around the page (if not in editing mode) are h, j, k, and l

  • h lets you move to the left
  • j lets you move down
  • k lets you move up
  • l lets you move to the right

How to save

  1. exit vim (mentioned above)
  2. git add (file name)
  3. git commit -m "(whatever you wanna write)"
  4. git push
